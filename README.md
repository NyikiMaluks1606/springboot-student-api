# Spring Boot Student API

Simple Spring Boot REST API built as part of Week 1 backend training.

## Features
- GET /students
- GET /students/{id}
- POST /students

## Tech Stack
- Java 17
- Spring Boot
- Maven

## How to Run
1. Clone the repo
2. Open in IntelliJ
3. Run the main application class
4. Visit http://localhost:8080/students
